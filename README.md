  ðŸ“± How It Works

  For Leaders:

  1. Visit the Leader Registration page
  2. Create your class account
  3. Get two links:
    - Sign-In Link - Keep on YOUR device only
    - Dashboard Link - View your class attendance
  4. During class, open your sign-in link and pass your device to each student

  For Students:

  1. Leader passes their device to you
  2. Choose "Returning Student" or "New Student"
  3. Fill out the form
  4. Submit - form auto-resets for the next person 

  For Admins:

  1. Use the Admin Dashboard to view all classes
  2. Filter by date, type, or search by name
  3. Export data to CSV
